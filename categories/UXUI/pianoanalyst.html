Piano Analytics
Analyse Comportementale UX

Définition

Piano Analytics est une plateforme d'analyse digitale développée par Piano (ex-AT Internet). Elle permet aux entreprises de mesurer, analyser et activer la donnée comportementale de leurs utilisateurs sur les sites web, applications mobiles et objets connectés, dans un cadre 100 % RGPD-compliant.

Objectifs

Mesurer le trafic : pages vues, clics, conversions, entonnoirs, rebonds…

Comprendre les comportements : segmentation fine, parcours utilisateurs, points de friction

Optimiser la performance : données e-commerce, KPI marketing, attribution multicanale

Respecter la vie privée :données first-party, tracking hybride (opt-in, opt-out, exempt)

Implementation

A)Tagging léger ("instant tracking")


html
                                
  (function(_cfg){
    var s = document.createElement("script");
    s.src = "https://tag.aticdn.net/piano-analytics.js";
    s.async = true; s.crossorigin = "anonymous";
    s.dataset.config = JSON.stringify(_cfg);
    document.head.appendChild(s);
  })({
    site: 123456789,
    collectDomain: "https://xxx.pa-cd.com",
    instantTracking: true
  });

→ Tracking de base immédiat

B)Envoyer un événement "page.display"


js

pa.sendEvent("page.display", {
  page: "Accueil",
  page_chapter1: "Cat1",
  page_chapter2: "Cat2",
  page_chapter3: "Cat3"
});

→ Événement + 4 propriétés contextuelles

C)Envoyer un événement de clic


js

pa.sendEvent("click.navigation", {
  click: "CTA Acheter",
  click_chapter1: "...",
  click_chapter2: "...",
  click_chapter3: "...",
  page: "Produit"
});

→ 5 propriétés incluant la page d'origine

Modèle de Données

Elément Description
Evénements page.display, click.navigation, transaction.*
Standard props > 70 événements, ~ 450 propriétés prédéfinies
Custom props Props définies par l'usager, actives/inactives, visibles/cachées
Processed props Envoyées dans le payload, dispo immédiatement
Calculated props Calculées en back-end, non envoyées dans le payload initial 

Measurements (données enrichies)

  • Usage : ajouter contexte (ex. nombre d'abonnés Facebook par jour)
  • Caractéristiques : numérique, temporel, agrégé
  • Enrichissement : propriétés additionnelles (ex. météo → température, bulletin)
  • Analyse : disponibles dans Explorer, Data Query, Workspaces (pas dans Data Flow)

Analyse & Extraction de données

  • Data Query : outil graphique / no-code pour créer segments/datasets/rapports
  • Data Mining : machine learning intégré, API, activation export
  • E-commerce : indicateurs transactionnels (CA, ARPPU, durée panier…)

Débug & QA

  • Network Tab : vérifiez les hits #xiti ou #vtag et le status 200
  • Tag Inspector : extension Chrome pour visualiser événements/propriétés
  • Stream Inspector : outil via Piano UI > Data Collection Portal > Outils
